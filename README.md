# nTraum's dotfiles

Personal dotfiles, feel free to take a look.
If you don't want to manage symlinks manually, take a look at [stow](https://www.gnu.org/software/stow/manual/stow.html).

## Usage (with stow)

```shell
$ cd dotfiles

# Create symlinks for zsh
$ stow zsh --target=$HOME
```
